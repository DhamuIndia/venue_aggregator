# MVP Scope

## Build First

- Public hall search and hall detail pages
- Hall owner onboarding and dashboard
- Admin approval for halls and vendors
- Image and video metadata with external storage URLs
- Booked-date and blocked-date calendar
- Customer enquiry and lead management
- Vendor onboarding and profile pages
- Vendor subscription plans and Razorpay integration
- Basic SEO pages for city, area, and category searches

## Avoid In MVP

- Microservices
- Advanced AI recommendations
- Real-time chat
- Mobile app before web validation
- Online hall booking payment unless required immediately
- Advanced bidding or package builder

## Roles

- Visitor or customer: search, view details, check availability, submit enquiries
- Hall owner: create hall profile, manage media, block dates, view leads
- Vendor: create vendor profile, subscribe to a plan, receive leads
- Admin: approve or reject listings, moderate content, track enquiries
- Super admin: manage plans, settings, roles, and audit controls

# Development Roadmap

## Phase 0: Product Planning

- Freeze MVP fields
- Confirm roles and permissions
- Prepare wireframes
- Prepare ER diagram
- Prepare API contract

## Phase 1: Backend Foundation

- Spring Boot application
- PostgreSQL connection
- Flyway migrations
- Auth, roles, and JWT
- Swagger/OpenAPI
- Actuator
- Docker development database

## Phase 2: Hall Listing

- Hall owner onboarding
- Hall profile CRUD
- Amenities, pricing, location, capacity, and rules
- Admin approval flow

## Phase 3: Media

- S3-compatible presigned upload flow
- Media metadata
- Primary image support
- Gallery ordering

## Phase 4: Availability

- Blocked dates
- Booked dates
- Availability API
- FullCalendar UI
- Double-booking prevention

## Phase 5: Enquiries

- Customer enquiry forms
- Lead assignment
- Owner/vendor dashboard
- Admin lead tracking

## Phase 6: Vendor Subscriptions

- Vendor onboarding
- Subscription plans
- Razorpay subscriptions
- Razorpay webhook handling
- Active and expired subscription state

## Phase 7: Search And SEO

- Location, capacity, budget, and amenity filters
- City, area, and category pages
- Sitemap
- Basic metadata

## Phase 8: Production Readiness

- Deployment pipeline
- Backups
- Object storage migration runbook
- Monitoring
- Logs
- Rate limiting
- Audit logs
- Security checks
